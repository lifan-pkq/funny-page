# 👍 大佬登场

一个有趣的恶搞网页，带点赞互动特效。

🔗 **在线预览**: https://lifan-pkq.github.io/funny-page/

## 特效功能

- 🎨 彩虹渐变边框动画
- 💫 点击照片触发表情雨
- 👆 悬浮放大+旋转效果
- 🔥 随机自动点赞（假装有人在看）
- 📱 移动端适配

## 本地运行

```bash
git clone https://github.com/lifan-pkq/funny-page.git
cd funny-page
open index.html
```

## 技术栈

- HTML5
- CSS3 (动画 + 渐变)
- 原生 JavaScript
